# Music Player App

React-based Music Player that allows users to search songs online, upload local music files, create playlists, and manage favorites.

## Features

- Online song search (iTunes API)
- Local song library
- Upload custom songs
- Playlist management
- Favorites section
- Animated album artwork
- Responsive React UI
- Modern glassmorphism design

## Tech Stack

- React.js
- Vite
- CSS3
- JavaScript
